## Introduction

WeatherAdaptive-Perception is a weather-aware object detection framework built on [YOLOv8](https://github.com/ultralytics/ultralytics). The core contribution is **EnvAdaptiveFusion**, a lightweight gating module that dynamically adjusts multi-scale feature fusion based on real-time weather conditions (clear, rainy, foggy, snowy, overcast).

The system inserts a **WeatherGatedConcat** mechanism at the FPN neck skip connections (P3/P4/P5), where per-level gate values are generated by a weather classification branch and applied to modulate the balance between backbone features and neck features before concatenation.

### Key Features

- **EnvAdaptiveFusion Module** — Weather-conditioned gating at FPN skip connections with interpretable per-level gate values (G_P3 / G_P4 / G_P5)
- **WeatherGatedConcat** — Learnable soft gate applied before feature concatenation, enabling adaptive emphasis on different scale features under varying weather
- **Weather Classification Branch** — Lightweight auxiliary classifier producing weather-condition logits used to drive gate generation
- **Video Inference Pipeline** — Adaptive frame extraction (Δn = ⌊f₀/fₛ⌋) with weather HUD overlay and real-time gate visualization
- **ONNX Export** — Export support with input shape `[1, 3, 640, 640]` and output shape `(1, 14, 8400)`

## Architecture

```
                        Input Image
                             │
                       ┌─────┴─────┐
                       │  Backbone  │
                       │ (YOLOv8)   │
                       └─────┬─────┘
                             │
                    ┌────────┼────────┐
                    P3       P4       P5  (Backbone feature maps)
                    │        │        │
              ┌─────┴──┐ ┌──┴───┐ ┌──┴───┐
              │Weather  │ │Weather│ │Weather│
              │Gated    │ │Gated  │ │Gated  │
              │Concat   │ │Concat │ │Concat │
              └─────┬──┘ └──┬───┘ └──┬───┘
                    │        │        │
                    ▼        ▼        ▼
              ┌─────────────────────────────┐
              │     FPN Neck (Modified)      │
              └──────────────┬──────────────┘
                             │
                   ┌─────────┼─────────┐
                   │         │         │
                 Small    Medium     Large
                 Head      Head      Head
                             │
                      Detection Output
                             
        ┌────────────────────────────────────┐
        │  Weather Classification Branch     │
        │  Input → Weather Logits → Gates    │
        │  G_P3, G_P4, G_P5 ∈ [0, 1]       │
        └────────────────────────────────────┘
```

## Installation

### Requirements

- Python >= 3.8
- PyTorch >= 2.0
- CUDA >= 11.7

### Setup

```bash
git clone https://github.com/Pengyu-Zhan/WeatherAdaptive-Perception.git
cd WeatherAdaptive-Perception
pip install -r requirements.txt
```

## Usage

### Training

```bash
python scripts/train.py --cfg configs/model/yolov8-envadaptive.yaml \
                        --data configs/data/bdd100k_weather.yaml \
                        --epochs 100 \
                        --batch-size 16
```

### Validation

```bash
python scripts/val.py --weights runs/train/best.pt \
                      --data configs/data/bdd100k_weather.yaml
```

### Video Inference

```bash
python scripts/video_inference.py --weights runs/train/best.pt \
                                  --source path/to/video.mp4 \
                                  --show-gates        # enable gate value HUD overlay
```

### ONNX Export

```bash
python scripts/export_onnx.py --weights runs/train/best.pt \
                              --imgsz 640
# Output: input [1, 3, 640, 640] → output (1, 14, 8400)
```

### Gate Visualization

```bash
python scripts/visualize_gates.py --weights runs/train/best.pt \
                                  --source path/to/images/ \
                                  --save-dir results/gate_vis/
```
